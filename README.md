# Autocallable Fixed Coupon Note Calculator

Production-ready Streamlit app for structured product pricing and scenario analysis.

## Features
- Multi-page navigation
- Input validation & session state
- Full payoff scenario analysis
- Sensitivity tables & charts
- CSV/Excel export
- Audit trail

## Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Product Logic
Models autocallable notes with:
- Early redemption triggers
- Coupon barriers
- Memory coupon feature
- Downside participation
- Call protection periods
